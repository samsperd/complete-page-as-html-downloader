Installation

Install the extension from Chrome Web Store.

The extension works within any browser that supports Chrome Web Store (like Edge, Vivaldi, etc.)

Setup

Pin the extension in Browser bar to make it available at all times on Browser toolbar. Otherwise, access the extension via the Extensions icon menu.

Recording Steps

Stepsy allows capturing the steps with a mouse click immediately. Just access the icon and click the Record a Guide icon.

Every mouse click will generate a screenshot. You will see the number of steps captured on the Stepsy icon in the toolbar:

Every screenshot will also contain a place of your mouse click.

You can see the preview of the last captured screenshot by accessing the Stepsy icon:

 

At any point in time, you can Pause and then Resume capturing via the same dialog.

Warning:

Recording works only within Browser windows.

  • It will not capture clicking on a tab header or Browser Toolbar.
  • It will not capture clicking on Browser system pages, like Settings or Chrome Web Store.

Masking PII data

If you need to hide any data within the browser before you start recording, use the blur icon within the recording popup. When you are done with blurring - just click the same icon.

Using the same icon during the recording, you will be able to turn on or off blur on top of all previously marked regions.

Editing screenshots

You can edit any of the captured screenshots in the history by clicking the edit icon on top of the screenshot preview panel. The popup window allows you to add elements like line, rectangle or highlight on top of the captured image.

Publishing the result

Stepsy can publish recorded steps as a Google Docs document on your Google Drive.

Select Finish and Share on the Stepsy dialog when you capture the steps.

If thats your first recording session after installation, Stepsy will ask you to authorize your Google account so that it knows Google Drive where to publish your document to.

FAQ

Whats the difference between Basic and Pro versions?

By default, after you authorize your Google Drive account, you are granted with a seven-day free trial of the Pro version. After seven days, your instance will become Basic.

Pro version advantages:

  • Stepsy automatically locates and merges duplicate screenshots so that you dont spend time deleting unnecessary steps yourself
  • Your screenshots dont have a small Stepsy watermark.
  • You can use Redact tool on images
